# Techa Site

Site institucional em React/Vite para apresentar a solucao e direcionar usuarios ao app.

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy no Vercel

Configure o projeto da Vercel com:

- Root directory: `frontend/site`
- Build command: `npm run build`
- Output directory: `dist`

Variaveis opcionais:

```env
VITE_API_BASE_URL=https://techa-backend.onrender.com
VITE_APP_URL=https://mobile-dusky-alpha.vercel.app
```
